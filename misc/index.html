colophon & about this site

This website is built using Middleman, a Ruby-based static site generator, and hosted on GitHub Pages under this repository. Text is set in the IBM Plex family of fonts.

I began putting this together around May 2018, but I got held up over the summer because of a few complications, so I returned in earnest in September, and the initial/beta version went live soon thereafter. Actually, it was his website that originally inspired me to use Middleman! Many thanks.

And thank you, too, for visiting! I hope you have a lovely time :)
